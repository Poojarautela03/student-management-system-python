# Student Management System (Python)

## About
A simple Python project to manage student records using JSON file storage.

## Features
- Add student
- View students
- Search student
- Update student
- Delete student
- Show topper
- Persistent storage using JSON

## Tech Stack
- Python
- JSON

## How to Run
1. Download the project
2. Run main.py
3. Use menu options

## Learning Outcome
- File handling
- Loops and conditions
- Dictionaries
- CRUD operations
