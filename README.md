# 🎓 Student Management System

> A lightweight Python application to manage student records with persistent JSON storage.

---

## 📌 About

A simple Python project to manage student records using JSON file storage.

---

## ✨ Features

| Feature | Description |
|---|---|
| ➕ Add Student | Register a new student record |
| 👁️ View Students | List all stored student records |
| 🔍 Search Student | Look up a student by name or ID |
| ✏️ Update Student | Modify existing student details |
| 🗑️ Delete Student | Remove a student from records |
| 🏆 Show Topper | Display the highest-performing student |
| 💾 Persistent Storage | All data saved using JSON file storage |
